# Taranga – AI-Powered Learning Difficulty Screening System

Taranga is a web-based AI-powered system designed to support early screening, analysis, and intervention for learning difficulties such as dyslexia, dyscalculia, dysgraphia, non-verbal learning disability, and auditory processing disorder.

## Project Status
Design and planning phase completed. Implementation will begin in the next milestone.

## Technology Stack
- Backend: FastAPI (Python)
- Frontend: ReactJS
- Database: PostgreSQL
- AI/ML: Scikit-learn, NLP techniques
- Version Control: Git & GitHub

## Repository Structure
- backend/ – API services and AI modules
- frontend/ – User interface
- documentation – Project reports and diagrams
